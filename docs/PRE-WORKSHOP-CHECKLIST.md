## Step 1 — Get Your Claude Subscription

Claude is the AI that powers your assistant. You need a Claude Max subscription.

1. Go to: **https://claude.ai**
2. Click **Sign Up** (or Log In if you already have an account)
3. Go to **Settings → Billing → Upgrade Plan**
4. Choose **Claude Max** — $100 USD/month

> **Why Claude Max?** The standard free plan does not include Claude Code — the version that can run on your computer and automate tasks. Claude Max unlocks the full power.

✅ Done when: You can log in to claude.ai and see "Max" next to your plan.

---

## Step 2 — Install VS Code

VS Code is the app where you will chat with your AI assistant. It is free.

**Mac:**
1. Go to: **https://code.visualstudio.com**
2. Click the big blue **Download for Mac** button
3. Open the downloaded file and drag VS Code to your Applications folder
4. Open VS Code from your Applications

**Windows:**
1. Go to: **https://code.visualstudio.com**
2. Click **Download for Windows**
3. Run the installer and click through (all default settings are fine)
4. Open VS Code

✅ Done when: VS Code opens and you see a welcome screen.

---

## Step 3 — Install the Claude Code Extension in VS Code

This connects Claude AI directly inside VS Code.

1. Open VS Code
2. Click the **Extensions icon** on the left sidebar (it looks like 4 squares)
3. In the search bar, type: **Claude Code**
4. Click **Install** on the result from Anthropic
5. A Claude icon will appear in your left sidebar

✅ Done when: You see the Claude icon in your VS Code sidebar.

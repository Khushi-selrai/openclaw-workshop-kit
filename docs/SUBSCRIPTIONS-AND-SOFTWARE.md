## If You Want an Always-On AI Server

Right now your assistant only runs when your laptop is open. If you want it running 24/7 — sending emails at night, monitoring leads while you sleep — you need a server.

| Option | Cost | Best For |
|---|---|---|
| **AWS Lightsail** | $5-$20 USD/month | Simple, reliable, beginner-friendly |
| **DigitalOcean Droplet** | $6-$12 USD/month | Simple setup, great documentation |
| **Hetzner VPS** | $4-$8 USD/month | Cheapest option, slightly more technical |
| **Vultr** | $6-$12 USD/month | Good performance, simple dashboard |

> **Luke's recommendation:** AWS Lightsail $10/month — simple to set up, always reliable. Ask Luke at the workshop if you want this — he can walk you through it in 10 minutes.

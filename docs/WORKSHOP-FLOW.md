# Workshop Flow Guide

## Goal
Create a seamless, high-impact Claude Code workshop experience.

---

## Recommended Flow

### Step 1: Setup
- Install Node.js
- Install Claude Code
- Login

### Step 2: First Success
- Run a simple prompt
- Show instant output

### Step 3: Guided Build
- Create a small system using Claude Code

### Step 4: Business Use Case
- Build a real-world solution (lead qualification, automation)

### Step 5: Optional Installs
- Telegram / WhatsApp / advanced integrations

---

## Key Principles

- Speed → get results fast
- Simplicity → avoid overwhelm
- Clarity → explain what’s happening
- Value → tie everything to business use

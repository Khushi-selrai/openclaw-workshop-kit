# OpenClaw Workshop — AI Business Assistant Kit

**Built by Selr AI — selrai.com.au**

This kit gives you everything you need to set up your own AI business assistant. Built for non-technical business owners at the OpenClaw Workshop.

---

## Quick Start (One Command)

Open Terminal and run:

```bash
git clone https://github.com/luke-selrai/openclaw-workshop-kit.git ~/workshop-kit && cd ~/workshop-kit && bash setup.sh
```

Then start your assistant:

```bash
cd ~/my-assistant && claude
```

Your assistant will greet you, ask about your business, and remember everything from that point on.
